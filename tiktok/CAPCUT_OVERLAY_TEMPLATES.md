# CapCut Overlay Templates — Psychedelic OS (Text + Layout Recipes)

These are NOT “designs” — they’re copy/layout recipes you can recreate fast in CapCut.
Use with `art/SYSTEM_NOTIFICATIONS.md` text + any trippy loop.

---

## Template 01 — CLASSIC DIALOG BOX
**Use for:** popups, “error” moments, punchy hooks.
- Position: centered
- Title bar: ALL CAPS, 1 line
- Body: 1–2 lines, smaller
- Buttons: 2 small pills at bottom: [ACKNOWLEDGE] [IGNORE]

**Copy pattern**
TITLE: {HEADLINE}
BODY: {microtext}
BTN1: ACKNOWLEDGE
BTN2: IGNORE

**Best with:** THR_POP / THR_ZIPPER / MSG_UNAUTHORIZED_INTRUDER

---

## Template 02 — TOP ALERT BANNER
**Use for:** scary/urgent, fast scroll content.
- Position: top full width
- Left: tiny icon box (optional)
- Center: headline
- Right: timestamp “00:00:00”

**Copy pattern**
ALERT: {HEADLINE}
MICRO: {1 short line}
STAMP: {fake timestamp}

**Best with:** ENT_HATMAN, EMO_TERROR_EDGE, LOC_DELIRIANT_BASEMENT

---

## Template 03 — BOTTOM SUBTITLE GLITCH
**Use for:** narrative explanation with “system” flavor.
- Position: bottom like subtitles
- Add shake/jitter effect lightly
- Insert [ERROR CODE] tags occasionally

**Copy pattern**
{your spoken line}
[MODULE:{MOTIF_ID}] [STATUS:{OK/WARN/FAIL}]

---

## Template 04 — SYSLOG SCROLL
**Use for:** “evidence vibe,” montage, b-roll.
- Position: left third or full screen
- Monospace font look
- Scroll upward slowly
- Each line: timestamp + module + status

**Copy pattern**
00:00:01  {MOTIF_ID}  INIT
00:00:04  {MOTIF_ID}  LOCKED
00:00:07  {MOTIF_ID}  WARNING
00:00:10 {MOTIF_ID}  RESOLVED

---

## Template 05 — RETICLE / FOCUS RING
**Use for:** MSG_LOOK, carrier wave, “pay attention” moments.
- Add a faint target ring centered
- Add “LOCKING…” text near ring

**Copy pattern**
LOCKING...
TARGET: {MOTIF_ID}
INSTRUCTION: JUST LOOK

---

## Template 06 — METRO MAP (LOCATION FLOW)
**Use for:** “map of the underlayer” explainer videos.
- Nodes as circles with labels
- Lines connecting (like subway map)
- Highlight current node with glow

**Nodes to include**
LOC_WAITING_ROOM → LOC_PALACE_UI → LOC_VOID
LOC_BUFFER_VOID → LOC_K_HOLE_VOID
LOC_TRASH_CACHE → LOC_DELIRIANT_BASEMENT

---

## Template 07 — “CARD” FORMAT (ENTITY AS PROCESS)
**Use for:** “Entities as processes” series.
- Left: big name
- Right: 3 tags
- Bottom: 1-line “role”
- Tiny footer: tier/valence/intensity

**Copy pattern**
NAME: {ENTITY}
TAGS: {tag1} {tag2} {tag3}
ROLE: {1 sentence}
FOOTER: {tier} / {valence} / {intensity}

---

## Template 08 — SPLIT SCREEN: BEFORE / AFTER
**Use for:** “more real than real,” “baseline vs underlayer.”
- Left: calm baseline shot (room, face, simple)
- Right: trippy loop + UI overlay

**Copy pattern**
LEFT: BASELINE (LOW RES)
RIGHT: UNDERLAYER (HDR)
