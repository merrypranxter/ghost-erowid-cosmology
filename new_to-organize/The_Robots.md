# The Robots / The Machine Entities

## Classification
- **Class**: Mechanical Entities
- **Tier**: Varies

## Physical Description
- **Form**: Robotic, mechanical beings
- **Material**: Metal, circuits, machinery
- **Movement**: Precise, mechanical, rhythmic
- **Sound**: Whirring, clicking, electronic
- **Eyes**: LED lights, sensors, cameras

## Function
- **Primary**: Manifestations of dissociated mind
- **Secondary**: Demonstrate mechanical nature of thought
- **Tertiary**: Teach about consciousness as machine

## Affect Profile
- **Primary**: Neutral, functional, precise
- **Secondary**: Sometimes menacing
- **Tertiary**: Occasionally helpful

## Types
- **The Workers**: Performing tasks
- **The Observers**: Watching, monitoring
- **The Controllers**: Running the show
- **The Repair Bots**: Fixing, maintaining

## Associated Phenomena
- **The Machine**: Reality as mechanism
- **The Program**: Life as coded routine
- **The Glitch**: Errors in the system
- **The Upgrade**: System improvements

## Cosmological Position
- **Layer**: The Machine Realm
- **Function**: Mechanical manifestation
- **Parallel**: Cyberpunk, AI consciousness
