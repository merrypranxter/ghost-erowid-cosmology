# The Folded Space / The Membrane

## Classification
- **Type**: Dimensional Space
- **Layer**: Interdimensional

## Description
Folded Space is the Salvia experience of reality as folded, 2D surfaces that can be turned like pages, unzipped like fabric, or creased like paper.

## Characteristics
- **Flatness**: Everything exists on 2D planes
- **Folding**: Space folds at angles
- **Pages**: Reality as book being turned
- **Edges**: Can see the "edge" of 3D space
- **Creases**: Lines where dimensions meet

## Phenomena
- **The Page Turn**: Reality as page being turned
- **The Zipper**: Unzipping space
- **The Fold**: Bending space at angles
- **The Edge**: Seeing beyond 3D

## Associated Entities
- Membrane Beings, The Wheel, The Lady Salvia

## Function
- **Primary**: Demonstrate true nature of space
- **Secondary**: Access to other dimensions
- **Tertiary**: Challenge 3D perception

## Cosmological Position
- **Layer**: The Membrane / Folded Dimension
- **Function**: Dimensional interface
- **Parallel**: String theory membranes, Flatland
