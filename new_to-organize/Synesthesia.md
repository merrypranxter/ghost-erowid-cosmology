# Synesthesia / Sensory Blending

## Classification
- **Type**: Sensory Phenomenon
- **Phase**: All LSD Phases

## Description
Synesthesia is the blending of senses—seeing sounds, hearing colors, tasting shapes. On LSD, this becomes a profound experience of sensory unity.

## Types
- **Chromesthesia**: Sound → Color
- **Spatial sequences**: Time → Space
- **Personification**: Numbers/letters → Personalities
- **Sound → Touch**: Hearing creates physical sensation
- **Taste → Vision**: Flavors create visual patterns

## Characteristics
- **Vivid**: More intense than normal perception
- **Consistent**: Same stimuli produce same cross-modal effects
- **Beautiful**: Often aesthetically pleasing
- **Revealing**: Shows underlying unity of senses

## Common Manifestations
- **Music → Visuals**: Songs create geometric landscapes
- **Voices → Colors**: Different voices have different colors
- **Touch → Sound**: Physical contact creates tones
- **Emotions → Shapes**: Feelings manifest as forms

## Function
- **Primary**: Reveal sensory unity
- **Secondary**: Enhance aesthetic experience
- **Tertiary**: Creative inspiration

## Technical Notes
- **Neural basis**: Cross-activation of sensory areas
- **Prevalence**: More common in psychedelic states
- **Permanent**: Some retain synesthesia after experience
