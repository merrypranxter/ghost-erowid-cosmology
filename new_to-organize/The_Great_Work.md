# The Great Work / Magnum Opus

## Classification
- **Type**: Cosmological Process
- **Domain**: Evolutionary Framework

## Core Concept
The Great Work is the ongoing process of cosmic evolution—the transformation of base matter into conscious spirit, the alchemical process on a universal scale.

## Stages
1. **Nigredo (Blackening)**: Dissolution, ego death, chaos
2. **Albedo (Whitening)**: Purification, clarity, insight
3. **Citrinitas (Yellowing)**: Illumination, wisdom, integration
4. **Rubedo (Reddening)**: Completion, union, transcendence

## DMT Correlation
- **Nigredo**: The terror, the void, ego dissolution
- **Albedo**: The light, the clarity, the insight
- **Citrinitas**: The download, the knowledge, the transformation
- **Rubedo**: The return, the integration, the new self

## Individual Application
- **Personal evolution**: Each life as alchemical process
- **Transformation**: Base self into enlightened self
- **Integration**: Making the spiritual material
- **Service**: Using transformation to help others

## Collective Application
- **Human evolution**: Species-level transformation
- **Planetary consciousness**: Earth as evolving organism
- **Cosmic evolution**: Universe becoming self-aware
- **The Work**: Ongoing process without end

## Alchemical Symbols
- **Philosopher's Stone**: Transmutation, perfection
- **Elixir of Life**: Immortality, wisdom
- **Prima Materia**: Raw substance of transformation
- **Coniunctio**: Sacred marriage of opposites

## Cosmological Position
- **Function**: The purpose of existence
- **Process**: Never-ending evolution
- **Goal**: Continuous becoming, not fixed state
