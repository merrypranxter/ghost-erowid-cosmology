# The Teachers / The Guides

## Classification
- **Class**: Teaching Entities
- **Tier**: High-Level

## Physical Description
- **Form**: Varies—often wise elder appearance
- **Material**: Light, energy, or ancient-looking
- **Presence**: Authority, wisdom, patience
- **Eyes**: Knowing, penetrating, kind
- **Movement**: Measured, deliberate, graceful

## Function
- **Primary**: Direct teaching and guidance
- **Secondary**: Answer questions, provide insight
- **Tertiary**: Facilitate transformation

## Teaching Methods
- **Direct transmission**: Information download
- **Socratic**: Questions leading to insight
- **Demonstration**: Showing rather than telling
- **Experience**: Learning through direct encounter

## Affect Profile
- **Primary**: Wise, patient, compassionate
- **Secondary**: Demanding, challenging
- **Tertiary**: Playful, trickster-like

## Associated Phenomena
- **The Lesson**: Specific teaching received
- **The Test**: Challenge to apply learning
- **The Gift**: Something given, to be used
- **The Task**: Assignment for after return

## Cosmological Position
- **Layer**: All layers, especially Peak Complex
- **Function**: Education, guidance
- **Parallel**: Gurus, mentors, wise elders
