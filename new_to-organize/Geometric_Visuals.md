# Geometric Visuals / Sacred Geometry

## Classification
- **Type**: Visual Phenomenon
- **Phase**: All phases

## Description
Geometric visuals are the fundamental building blocks of DMT visual phenomena—intricate, hyper-detailed patterns that underlie and compose all other visual elements.

## Pattern Types
- **Fractals**: Self-similar patterns at all scales
- **Tessellations**: Repeating tile patterns
- **Lattices**: Grid-like structures
- **Mandalas**: Circular, symmetrical patterns
- **Kaleidoscopes**: Mirror-symmetric patterns

## Common Forms
- **Hexagons**: Six-sided, bee-hive like
- **Triangles**: Three-sided, pyramidal
- **Squares**: Four-sided, grid-like
- **Circles**: Curved, mandala-like
- **Spirals**: Curved, expanding/contracting

## Characteristics
- **Detail**: Hyper-detailed, infinite complexity
- **Color**: Saturated, glowing, iridescent
- **Movement**: Pulsing, breathing, rotating
- **Depth**: 3D, sometimes 4D (time as dimension)

## Function
- **Primary**: Visual language of the other side
- **Secondary**: Information encoding
- **Tertiary**: Beauty as communication

## Technical Notes
- **Mathematical basis**: Many patterns correspond to known mathematical structures
- **Cross-cultural**: Similar patterns appear in all cultures' sacred art
- **Neural basis**: May reflect visual cortex architecture
