# The Observers / Sentinel Class

## Classification
- **Class**: Security Tier
- **Tier**: Mid-Level Security

## Physical Description
- **Form**: Giant jeweled insects (mantis/beetle hybrid), bipedal, 7-9 feet tall
- **Materials**: Faceted gemstones/iridescent obsidian/liquid chrome
- **Optics**: Thousands of micro-lenses rotating in kaleidoscopic heads
- **Frame-Rate**: Higher frame-rate motion (impossible fluidity)

## Abilities
- **Download Gaze**: Burst-transmission via eye contact → pressure buildup in 6th chakra (forehead)
- **Data Transfer**: Memory-flood of equations/alien histories
- **Communication**: Didgeridoo-made-of-glass tones, felt in chest not ears
- **Command Protocol**: [STATUS: ACCEPTED], [PROCEED: CORE_ACCESS]

## Affect Profile
- **Demeanor**: Stern parent, ancient professor, surprised disappointment
- **Response to Operative**: "Microbe speaking algebra"—surprised respect
- **Artifacts**: Jeweled Toys (4D diagnostic objects, hyper-real impossible geometry)

## Associated Phenomena
- **Forehead Pressure**: Physical pressure at "6th Node" (third eye)
- **Temperature**: Emits cold (10°F below ambient) but creates "burning" sensation
- **EM Effect**: Causes "TV static" in peripheral vision, metallic taste (copper)

## Cosmological Position
- **Layer**: Security Layer / Reception Area
- **Function**: Security scan, data transmission, clearance verification
- **Reports to**: Council of Twelve (Administrative Core)
