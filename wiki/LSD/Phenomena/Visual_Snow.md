# Visual Snow / Static Vision

## Classification
- **Type**: Visual Phenomenon
- **Phase**: All LSD Phases

## Description
Visual snow is the perception of a fine, static-like overlay on vision—like seeing the world through a field of tiny, moving particles.

## Characteristics
- **Appearance**: Fine dots, particles, static
- **Color**: Usually white or colored
- **Motion**: Constant subtle movement
- **Density**: Varies, can be thick or sparse
- **Location**: Overlay on normal vision

## Variations
- **White snow**: Classic TV static appearance
- **Colored snow**: Rainbow or specific colors
- **Patterned**: Geometric arrangements
- **Intensified**: More pronounced on LSD

## Associated Phenomena
- **Pattern recognition**: Seeing patterns in static
- **Breathing walls**: Surfaces appear to move
- **Color enhancement**: Colors more vivid
- **Tracers**: Motion trails

## Scientific Understanding
- **Neural basis**: Visual cortex hyperactivity
- **Persistence**: Can occur without drugs (HPPD)
- **Normal variation**: Some have it naturally
- **LSD effect**: Enhances existing tendency

## Function
- **Primary**: Visual system activation
- **Secondary**: Pattern recognition enhancement
- **Tertiary**: Aesthetic effect

## Management
- **Acceptance**: Usually not problematic
- **Focus**: Look beyond it to see clearly
- **Lighting**: Affected by light conditions
- **Integration**: Usually fades with time
