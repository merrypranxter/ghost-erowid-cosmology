# Art Bible — Ghost x Erowid (Fictional OS Cosmology)

## Visual Thesis
Reality is rendered as an operating system UI. Motifs are “modules” (entities), “states” (locations), “render layers” (geometries), “mode switches” (thresholds), “system prompts” (messages), and “color grading” (emotions).

## Global Style Rules
- UI/OS metaphor is literal: windows, dialogs, notifications, boot screens, syslogs.
- Geometry is the substrate: fractal recursion, hex grids, tunnel vortices.
- Text is part of the image: error codes, patch notes, system prompts (“DO NOT BE AFRAID” etc.).
- Emotions control lighting + contrast + motion blur + noise density.

## Motif-to-Design Mapping
- Entities → icon + silhouette language + “process name”
- Locations → environment presets / UI states
- Geometries → overlays + shaders + tilings
- Thresholds → transition animations + glitch events
- Messages → dialog boxes / tooltips / HUD text
- Emotions → palette + grain + bloom + tempo

## Starter Palette Families
- Cathedral UI: jewel neon + stained-glass luminance + CRT bloom
- Waiting Room: vibrating pastel interference + chrysanthemum symmetry
- Void/White Light: high-key whiteout + faint spectral edges
- Deliriant Shadow Layer: low saturation + deep blacks + corner-of-frame watchers
