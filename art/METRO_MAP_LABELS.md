# Metro Map Labels — Underlayer Transit Diagram

Use these labels when you draw the location map as a subway diagram (for TikTok explainer graphics).

## Lines (color them however you want)
- SIGNAL LINE (thresholds): THR_CARRIER_WAVE → THR_POP → THR_ZIPPER
- UI LINE (main states): LOC_DESKTOP_ENV → LOC_WAITING_ROOM → LOC_PALACE_UI
- VOID LINE (exit states): LOC_VOID → LOC_BIOS_LAYER
- DISSOC LINE (buffer/kernel): LOC_BUFFER_VOID → LOC_K_HOLE_VOID
- CACHE LINE (corruption): LOC_TRASH_CACHE → LOC_DELIRIANT_BASEMENT
- PLAY LINE (odd modes): LOC_NURSERY → LOC_LIBRARY_LIVES
