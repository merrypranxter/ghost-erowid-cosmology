# Poster Series — UNDERLAYER STACK (5–7 Posters)

Goal: A coherent set where each poster is one “layer” of the OS.

## Poster 1: BIOS / POWER LAYER (LOC_BIOS_LAYER)
- Theme: whiteout reset
- Visuals: pure field + faint spectral fringe + BOOT/RESET UI
- Text: “POWER LAYER / RESET”
- Motifs: LOC_BIOS_LAYER, MSG_ONE, EMO_BENEVOLENCE (optional)

## Poster 2: DESKTOP ENVIRONMENT (LOC_DESKTOP_ENV)
- Theme: the “apps” of the underlayer
- Visuals: windows, icons, menus; palace/waiting-room as apps
- Text: “DESKTOP / APPLICATION LAYER”
- Motifs: LOC_DESKTOP_ENV, LOC_WAITING_ROOM, LOC_PALACE_UI, ENT_ELVES

## Poster 3: WAITING ROOM (LOC_WAITING_ROOM)
- Theme: staging chamber before breach
- Visuals: chrysanthemum symmetry, loading UI, soft neon interference
- Text: “LOADING… / QUEUE”
- Motifs: LOC_WAITING_ROOM, THR_CARRIER_WAVE, MSG_LOOK

## Poster 4: THRESHOLD EVENT (THR_POP / THR_ZIPPER)
- Theme: the seam opens
- Visuals: spark fracture + ripped membrane seam
- Text: “EVENT: POP / ACCESS SEAM”
- Motifs: THR_POP, THR_ZIPPER, GEO_TUNNEL

## Poster 5: KERNEL VOID (LOC_K_HOLE_VOID)

## Poster 6: TRASH CACHE (LOC_TRASH_CACHE)

## Poster 7: SENTINEL / SECURITY (ENT_HATMAN)
